# Final Project

The final project is in the `final` folder.

All the requirements are met:

- `final.py` is included.
- The program has a menu.
- The program reads from and writes to a text file for the high score.
- The program is broken into several functions.
- Some functions return values and can be unit-tested.
- Unit tests are included in `test_final.py`.
- The game keeps running until the player chooses to quit.

Project status: complete.

Run the game from the final folder:

```bash
cd final
python3 final.py
```

Run the tests from the final folder:

```bash
python3 test_final.py
```
